视频混剪工具 - Windows 应用
署名：JackLuo

启动方式：
1. 解压整个 zip。
2. 双击 VideoMixerUI.exe。
3. 浏览器会自动打开 http://127.0.0.1:8787。

功能：
- 左侧显示已选择视频、预计可用片段、计划生成成片。
- “每个源视频末尾排除秒数”：填入 N 后，每个源视频最后 N 秒不会进入切分和拼接。
- “输出位置”旁边的“选择”按钮可以打开文件夹选择窗口。
- “批量片头片尾”：批量选择主体视频，并给每个视频统一添加一段片头、一段片尾。
- 片头和片尾可以只选其中一个，也可以两个都选。

关闭服务：
- 双击“关闭混剪服务.bat”。

注意：
- 需要系统已安装 FFmpeg。
- 如果文件夹里有 _internal 目录，请不要删除，也不要只单独复制 exe。
